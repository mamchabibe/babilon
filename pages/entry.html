Register by group, not by player

This form creates one team account in Supabase Auth. The group name, team email, and player names are saved together, while the shared password controls access to the account.

  • Use one group email that your team can access when needed.
  • Add all player names in the team so the organizers know who belongs to the account.
  • After registration, teammates can use the same account credentials to enter the site.
Before this works live, add your Supabase project URL and anon key in assets/js/supabase-config.js, then run the SQL file in supabase/teams_schema.sql.
